Features

Everything you need. Nothing to install.

A complete GDS layout environment that runs in your browser — with AI that understands what you want to build.

🤖

AI-Powered Design

Describe shapes in plain English — "create a 5×5 array of 3μm circles on layer 0" — and the AI generates perfect GDS geometry instantly.

⬡

GDS & OASIS Support

Open, edit and export .gds and .oas files. Full support for boundaries, paths, cell arrays (AREF), references (SREF) and text elements.

✏️

Draw Tools

Rectangle, circle, polygon and path tools with snap-to-grid, real-time preview and precise coordinate display in nm and μm.

◎

Measure & Inspect

Click any shape to measure diameter and pitch. Save measurements as persistent rulers on the canvas.

🔍

DRC Checks

Run design rule checks for minimum spacing and feature size violations directly in the browser.

⌀

Wafer Template

Overlay a wafer outline at any diameter. Toggle dark-field / light-field polarity for mask ordering specifications.

↩

Undo / Redo

Full 30-step history. Select, move, rotate, mirror, copy and delete elements. Arrow-key nudging supported.

📷

PNG Export

Export a screenshot of your canvas — perfect for papers, presentations, and fabrication documentation.

✛

Alignment Marks

Insert standard cruciform alignment marks at the origin — essential for multi-layer mask sets.

File Formats

Import & export industry standards

Full read and write support for both major layout formats used in semiconductor and MEMS fabrication.

GDS / GDSII

GDSII Stream Format

The industry-standard format for IC and mask layout. Supports hierarchical cell arrays, paths, boundaries and text labels.

IC designPhotonicsE-beam lithographyFoundry submission
OAS / OASIS

OASIS Format

The modern successor to GDSII. Smaller files, native circle support and efficient array encoding. Compatible with advanced fabs.

Advanced nodesLarge layoutsMEMSMicrofluidics
